# Persona: Marcus — The Improver

## Profile
- **Name**: Marcus
- **Age**: 38
- **Handicap**: 14
- **Plays**: 2-3 rounds/month + cage 2-3×/week
- **Goals**: Get to single digits within 2 years
- **Pain points**: Inconsistent contact, slice on driver, wedge distance control
- **Tech comfort**: High — uses GPS apps, swing analyzers, follows golf YouTube
- **Voice with Kevin**: Coach mode primary; wants technical insights; will spend
  time reading analysis details

## Mental model
Marcus opens **SwingLab first** when launching the app. Logs cage sessions
methodically. Reads Phase K analysis carefully. Tests recommended drills.
Plays rounds expecting to apply what he practiced. Asks Kevin "what should I
be working on" voice queries. Reviews recap analytically.

## Critical needs from product
- **Cage analysis quality is paramount** (broken Phase K = broken product for him)
- **Drill library depth** matters
- **Round → practice connection** meaningful (does today's miss tendency match
  what cage said last week?)
- **Stats trend over time**
- **Persistent context shows up** (Kevin remembers his patterns)
- **TightLie and SmartVision genuine value** — won't tolerate gimmicks

## Failure modes that lose Marcus
- Phase K produces low-confidence analysis on most uploads
- Drills feel generic, not connected to his specific issue
- Rounds don't reference recent practice
- No multi-round trend visible
- Kevin treats every session as fresh, doesn't build understanding
- SmartVision feels like a toy (not a planning tool)

# SmartPlay Caddie — User Quick-Start Guide

## Your Caddie Team
Three personas. Switch any time.
- **Kevin** — balanced
- **Tank** — direct, no-fluff
- **Serena** — analytical, calm

Voice: *"Switch to Tank"* / *"Switch to Serena"* / *"Switch back to Kevin"*

## Five Tabs at the Bottom
- **Play** — pick course, set up round, start playing
- **Caddie** — talk to your caddie, manage the round
- **SwingLab** — practice tools + swing video
- **Scorecard** — current round scorecard
- **Dashboard** — stats, recent rounds, handicap

## Starting a Round
1. Play tab → search or tap a course
2. Set round mode (Break 100 / 90 / 80 / Free Play)
3. Optionally add notes (tap mic icon to dictate)
4. Tap **Start Round** → routes to Caddie tab

## Talking to Your Caddie
- **Push-to-talk**: tap mic / avatar on Caddie tab
- **Hands-free**: Settings → Voice → Active Listening → ON

Try:
- *"What's my score?"*
- *"What hole am I on?"*
- *"Open SmartVision"*
- *"Cart mode on"*
- *"Switch to dark mode"*

## SmartVision (hole preview)
- Caddie tab → Tools → SmartVision (or voice)
- T = tee, Y = your target/layup, P = pin
- Drag Y to set layup; carry yardage shows
- F/M/B = front/middle/back of green yardages

## Spectator Hero Shot
1. Say *"Watch this"* or *"Check this out"*
2. Camera records ~5 seconds
3. Replay + Share buttons appear
4. Hit **Share** → text / AirDrop / social

## PuttWatch (with Meta Ray-Ban glasses)
1. Say *"Watch this putt"* or *"Watch this chip"*
2. Caddie acknowledges
3. You: *"Hey Meta, record a video"*
4. Putt
5. Later → SwingLab → Upload → pick clip → tag Putt/Chip → analysis runs

## Cart Mode
Riding instead of walking? Tell the caddie:
- *"Cart mode on"* — tightens shot detection for cart play
- *"Cart mode off"* or *"Walking mode"* — back to default
- Or: Settings → Round Experience → Riding in a cart

## When the Caddie Misses a Shot
Tools menu → Log shot:
1. Tap **↺ Use fresh GPS fix** → pins exact location
2. Pick hole (override if wrong)
3. Pick club, outcome, direction
4. Save

## Ending a Round
- **Save & end** — locks round, updates handicap, generates recap
- **Discard** — full reset, no record

## Reporting a Bug
Say: *"Tank, log this — [what's wrong]"*
Lands in Settings → Owner Tools → Issue Log.

## SwingLab Quick Reference
- **Drills** — primary issue catalog + pro videos
- **Range Mode** — multi-shot range/studio/cage session
- **SmartMotion** — single swing, instant analysis
- **Arena** — bag distances, tempo trainer, putting clock
- **Swing Library** — all your swings with body overlay + swing trace
- **Acoustic Test Bench** — validate strike detection

## Settings Highlights
- Caddie Team — pick personas
- Trust Spectrum (Kevin's presence) — Quiet → Full
- Cart Mode toggle
- Active Listening toggle
- Voice on Phone Speaker (default ON)
- Theme — Light / Dark / System
- Owner Tools → Issue Log
